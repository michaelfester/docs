The Go to Tab action allows you to move the user to a specific tab in your app.

This is different from the [Show New Screen](/docs/show-new-screen) action which allows you to link to a sheet and customize that screen however you like. With Navigate to Tab, you have to already have created a Tab so the action can move the user there.
