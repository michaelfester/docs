The Compose Email action allows users to tap a [Component](/docs/components) and trigger an email draft on the user's device.

![](/docs/reference/actions/compose-email/Group%2060.png)

After you've added it, make sure the component is bound to the column that your emails are in.

If you want to be able to configure mailto properties like `Subject` `CC` `BCC` and `Body` then you should use the [**Email Component**](/docs/email-component).

{% callout title="" %}
You can enable actions like Send Email on a number of components like the [Button](docs/button-component) and [Text](docs/text-component) components.
{% /callout %}
