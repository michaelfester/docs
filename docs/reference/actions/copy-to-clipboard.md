The Copy to Clipboard action copies a value to the user's clipboard.

{% figure src="/docs/reference/actions/copy-to-clipboard/copy-to-clipboard.png" caption="Configure Copy to Clipboard" /%}

## Data

Once you configure the Copy to Clipboard action, you need to set the data source to be copied. This can be any of the following:

1. Custom Text
2. Unique values like [Current Date/Time](/docs/current-date-time), a [Unique Identifier](/docs/unique-id), [User's Email Address](/docs/users-email-address), or the [Link to the current screen](/docs/deep-links)
3. Data from a specified column in your sheet

{% figure src="/docs/reference/actions/copy-to-clipboard/copy-to-clipboard-custom.png" caption="Custom Values for Copy to Clipboard" /%}

{% callout title="" %}
You can enable actions like Copy to Clipboard on a number of components like the [Button](/docs/button) and [Action Text components](/docs/action-text).
{% /callout %}
