Glide Pages are now Glide Apps. Please refer to [this article](/docs/meet-the-new-glide-apps) for up-to-date information, as some of this documentation is outdated.
{% /callout %}

We can trigger an app to show the user's profile screen with an Action. First, we have to choose a [Component](/docs/components) where we can add an action, then we select the action Show User Profile Screen and when clicked, it will redirect your user to their User Profile Screen.
