The Open Link action allows you to link a [Button](/docs/button-component), [Action Text](/docs/action-text) or [Image](/docs/image-component) to an external webpage.

{% figure src="/docs/reference/actions/open-link/setting-up-open-link.png" caption="Open Link configuration" /%}

You can set this up by configuring the Action Setting for your element and set it to Open Link. Once you've set the action, choose the column that contains your target links.

We can choose the URL to be custom or from a column in our database. Also, we can change the Target. We can select Browser to open the link in a new tab, or Web View to open it within our app
