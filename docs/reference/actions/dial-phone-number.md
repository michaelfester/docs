The Dial Phone Number action allows you to trigger a phone call when the item is tapped.

![](/docs/reference/actions/dial-a-phone-number/2020-07-19%2021.33.28.gif)

You can enable the action on a component like the [Button](/docs/button-component) component. After you've added it, connect a column that has valid phone numbers in it and then add the `Phone call` action.
