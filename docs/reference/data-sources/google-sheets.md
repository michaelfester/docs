[Google Sheets](https://sheets.google.com/) is a cloud-based spreadsheet program very similar to Excel. You can choose a spreadsheet from your Google account to serve as the data source for your Glide project.

{% callout %}
If you created your Glide account using your work email, you must create an associated Google account to integrate Google Sheets with Glide. To learn how to create a Google account using an email type that isn’t Gmail, see [Create a Google Account](https://support.google.com/accounts/answer/27441?hl=en#existingemail).
{% /callout %}

## How to use Google Sheets with Glide

1. In the Layout Editor, select **New project**.
2. Name your project, select **App** or **Page**, then **Continue**.
3. Select **Google Sheets** as the data source, then **Continue**.
   ![Image shows the modal to select a source with Google Sheets as the focus among all data source options](/docs/reference/data-sources/google-sheets/choosing-google-sheets-for-your-project.png)
4. Sign in to your Google account.
5. When prompted, select the file from Google Drive to use as your project’s data source, then **Select**.
   {% callout %}
   If cookies are enabled in your browser, Google sign-in is only necessary the first time you use your Google account to create a Glide project.
   {% /callout %}

Glide creates a starter project using your Google spreadsheet as the data source.
