{% callout title="Meet the new Glide Apps" %}
Glide Pages are now Glide Apps. Please refer to [this article](/docs/meet-the-new-glide-apps) for up-to-date information, as some of this documentation is outdated.
{% /callout %}

Push notifications for chat messages and comments are supported on Android but not iOS yet. If you have an app installed on your Android phone you will get notifications when:

- Another user comments on a post that you have commented on, you will receive a notification
- Another user chats in the chat tab, you will get a notification.

{% figure src="/docs/reference/faq/notifications/push-notification-android.png" caption="Push notification on Android" /%}

{% callout title="Notification settings" %}
Make sure your settings are configured properly to receive push notifications on your Android device.
{% /callout %}
