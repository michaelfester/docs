**The Unique Identifier** is a special value that you can assign to a row when users add, edit, or submit forms. This unique identifier gives you additional control over relations between items in your app.

![](/docs/reference/values/unique-identifier/uniqueid.gif)

## Configuration

The Unique Identifier can only be added to screens where users can add, edit, or submit new entries. You can start by adding the Unique Identifier component. After this, choose the column where the unique values will be stored.

![](/docs/reference/values/unique-identifier/uniqueid.png)
