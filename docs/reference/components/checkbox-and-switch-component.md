The Switch and Checkbox components allow you to toggle boolean values in your data source.

{% figure src="/docs/reference/components/checkbox-and-switch-components/use-case-of-a-switch-component-in-pages.png" caption="" /%}

## Data

Select a column in your data source to write the value to.

## Design

You can use the **Label** and **Description** fields to provide direction to your users.

## Text Style

In Apps, you can enable the **Allow text wrapping** option for longer labels or descriptions. This wraps the text to the next line on the screen.
