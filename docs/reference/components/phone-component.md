{% callout title="" %}
The Phone Component is only available in Glide [Apps](/docs/introduction-to-apps), not Glide [Pages](/docs/introduction-to-pages).
{% /callout %}

The Phone component creates a clickable link that allows users to trigger a text or phone call on their device.

{% figure src="/docs/reference/components/phone-component/use-case-of-the-phone-component.png" caption="" /%}

## Configuration

Once you’ve added the Phone component, you have to make sure to link that to the column that contains the target phone number. You can also use a [Custom Value](/docs/custom-values) for the phone number.

The component allows both a Message and a Call function. You can optionally pre-populate the message body. This can either be custom text or a column value.

{% figure src="/docs/reference/components/phone-component/configuring-message.png" caption="" /%}

You also have the option to turn on/off the Text Message functionality.

{% figure src="/docs/reference/components/phone-component/turn-on-off-text-message.png" caption="" /%}
