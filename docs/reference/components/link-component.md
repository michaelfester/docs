## Configure a Link component

### Data

In the **General** tab on the right side of the Layout Editor, locate the **Data** section. Use the menu in **Link to** to select the value from your data to display, or select **Custom** to enter a [Custom Value](/docs/custom-values)for this field.

### Design

In the **General** tab on the right side of the Layout Editor, locate the **Design** section.

- Configure the **Style** of the link.

{% callout title="" %}
**Compact** will display the link as a button with a width that adjusts depending on the chosen **Title;
Row** will display the link in a full-width row along with an optional **Caption** field.
{% /callout %}

- Use the menu in **Title** to select the value from your data to display, or select **Custom** to enter a custom value for this field. The Title can also be left blank if you only want to display an **Icon**.

- If using the **Row** style, use the menu in **Caption** to select the value from your data to display or select **Custom** to enter a custom value for this field.
