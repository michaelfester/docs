{% callout title="" %}
The Link Component is only available in Glide [Apps](/docs/introduction-to-apps), not Glide [Pages](/docs/introduction-to-pages).
{% /callout %}

## Configure a Link component

### Data

In the **General** tab on the right side of the Layout Editor, locate the **Data** section. Use the menu in **Link** to select the value from your data to display, or select **Custom** to enter a [Custom Value](/docs/custom-values) for this field.

### Design

In the **General** tab on the right side of the Layout Editor, locate the **Design** section.

- Configure the **Label** to display the link's **Page title**, **Last path component**, **Short URL**, **Full URL**, or **Caption as title**

{% callout title="" %}
If **Caption as title** is selected, the label is hidden and the **Title** field is used to display the link's label.
{% /callout %}

- Use the menu in **Title** to select the value from your data to display, or select **Custom** to enter a Custom Value for this field. The Title can also be left blank if you only want to display the Label.
