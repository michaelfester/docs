{% callout title="" %}
The Action Text Component is only available in Glide [Apps](/docs/introduction-to-apps), not Glide [Pages](/docs/introduction-to-pages).
{% /callout %}

## Data

On the right side of the Layout Editor in **General > Data > Text**, use the menu to select the value that will be displayed for the user. This can be custom text or the data from a selected column.

{% figure src="/docs/reference/components/action-text/image%20(2).png" caption="Selecting a value that will be displayed for the user" /%}

{% callout title="" %}
Glide presents different configuration options tailored to each action. For detailed information about all of the actions available in Glide, check out Actions.
{% /callout %}

## Design

### Title

Glide automatically uses the column title from the data you select for the Action Text component. You can change the displayed title for the component in the **Title** property, located on the right side of the **Layout Editor**.

You can also type a custom title or choose to leave it empty.

{% figure src="/docs/reference/components/action-text/image%20(3).png" caption="Adding a title" /%}

### Icon

You can display an icon with your Action Text component. The default setting for an icon is **None**. You can select an icon, and Glide will add it to the design of your Action Text component.

{% figure src="/docs/reference/components/action-text/image%20(4).png" caption="Adding an icon" /%}

## Options

You can limit the amount of text that is displayed for your Action Text component by selecting this option in **General > Options**.

Users can tap **more** to view the full content.

{% figure src="/docs/reference/components/action-text/image%20(5).png" caption="Limiting the amount of text that is displayed" /%}

## Actions

This is the behavior that will be triggered when users tap on the text. You can choose from any of the actions available in Glide, or you can **create a new action**.

1. In **General > Action**, select the menu arrows.
2. In **Configure Action**, select the dropdown menu to view action options.
3. Select the action to be triggered when users tap your Action Text component.
4. If prompted, apply any additional settings in **Configure Action**.

{% figure src="/docs/reference/components/action-text/image%20(6).png" caption="Configuring Show Detail Screen action" /%}
